## Задача
Загрузи контекст проекта Theatre Management System для эффективной работы.

## Выполни:

1. Прочитай основные файлы:
   - `CLAUDE.md` — краткий контекст и команды
   - `.claude/memory-bank/00_PROJECT_INSTRUCTIONS.md` — главные инструкции
   - `.claude/memory-bank/01_PROJECT_CONTEXT.md` — бизнес-требования
   - `.claude/memory-bank/02_ARCHITECTURE.md` — архитектура

2. Если есть план работ, прочитай:
   - `docs/refactor-plan.md` — текущий план переработки
   - `docs/decisions.md` — принятые решения

3. Определи текущую фазу разработки

После прочтения:
- Подтверди что понял структуру проекта
- Укажи текущую фазу разработки из roadmap
- Перечисли критические проблемы для исправления
